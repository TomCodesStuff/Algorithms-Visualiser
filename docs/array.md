# Array Class Documentation 

Below are the helper functions provided by the `Array` base class. 


| Function:           | Parameters:                   | Return Value:                          | Description:                                                                        | 
| --------------------| ----------------------------- | -------------------------------------- | ----------------------------------------------------------------------------------- |
| get                 |                               | An array of Integers                   | Returns the entire array.                                                           |
| getBarColours       |                               | An array of Strings                    | Returns an array containing the colour of each bar displayed on screen.             |
| append              | value : int                   |                                        | Adds the passed value to the end of the array.                                      |
| pop                 |                               |                                        | Removes the last value from the array.                                              |
| sort                |                               |                                        | Sorts the array in ascending order.                                                 |
| shuffle             |                               |                                        | Shuffles the array.                                                                 |
| setAt               | index : int <br> value : int  |                                        | Replaces the value at the passed index with the passed value.                       |
| getAt               | index : int                   | An Integer.                            | Returns the value at the passed index.                                              |
| setColourAt         | index : int <br> value : str  |                                        | Replaces the colour at the passed index with the passed colour.                     |
| getColourAt         | index : int                   | A String.                              | Returns the colour at the passed index.                                             |
| resetBarColours     |                               |                                        | Sets all bar colours to Black                                                       |
| size                |                               | An Integer                             | Returns the arrays length. NOTE: Pythons len() function is also supported.          |
| getMin              |                               | An Integer                             | Returns the smallest element in the array                                           |
| getMax              |                               | An Integer                             | Returns the biggest element in the array                                            |
| isInArray           | value : int                   | A Boolean (True/False)                 | Returns True if the passed value is in the Array. Otherwise False is returned       |
