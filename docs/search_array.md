# SearchArray Class Documentation 

Below are the helper functions provided by the `SearchArray` class. <br>
`SearchArray` inherits from the `Array` class. The methods provided by the `Array` base class are documented [here.](./array.md)  


| Function:           | Parameters:                   | Return Value:                          | Description:                                                                        | 
| --------------------| ----------------------------- | -------------------------------------- | ----------------------------------------------------------------------------------- |
| setTarget           |  value : int                  |                                        | Sets the search target to the passed value.                                         |
| getTarget           |                               | An Integer                             | Returns the target algorithms search for.                                           |
