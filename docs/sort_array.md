# SearchArray Class Documentation 

Below are the helper functions provided by the `SortArray` class. <br>
`SortArray` inherits from the `Array` class. The methods provided by the `Array` base class are documented [here.](./array.md)  


| Function:           | Parameters:                   | Return Value:                          | Description:                                                                              | 
| --------------------| ----------------------------- | -------------------------------------- | ----------------------------------------------------------------------------------------- |
| isSortAcending      |                               | A Boolean (True/False)                 | Returns True if the sort direction is ascending. Otherwise False is returned.             |
| isSortDescending    |                               | A Boolean (True/False)                 | Returns True if the sort direction is descending. Otherwise False is returned.            |
| swapAt              |  i : int <br> j : int         |                                        | Swaps the values at the two passed indexes                                                |
| swapColoursAt       |  i : int <br> j : int         |                                        | Swaps the colours at the two passed indexes                                               |
| isSwapNeeded        |  i : int <br> j : int         | A Boolean (True/False)                 | Returns True if two values need to be swapped depending on the sort direction.            |
| areElementsEqual    |  i : int <br> j : int         | A Boolean (True/False)                 | Returns True if the values at the passed indexes are equal. Otherwise False is returned   |
